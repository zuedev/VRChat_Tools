About

VRChat Tools is a powerful command-line interface for managing your VRChat avatars. List, view, and interactively delete avatars with ease, all through the official VRChat API. Features persistent authentication and supports both npm and Docker installation.

Installation

Docker (Recommended for Isolated Usage)

# Pull the latest image
docker pull ghcr.io/zuedev/vrchat_tools:latest

# Run with volume mount
docker run -it --rm \
  -v vrchat-data:/data \
  ghcr.io/zuedev/vrchat_tools:latest avatars

Global Installation (Recommended)

# Install globally via npm
npm install -g vrchat-tools

# Or from GitHub Packages
npm install -g @zuedev/vrchat-tools --registry=https://npm.pkg.github.com

Local Development

npm install

Usage

List All Avatars

vrchat-tools avatars

Get Current Avatar

vrchat-tools avatars current

Delete Avatars Interactively

vrchat-tools avatars delete

Press D to delete the current avatar, or Enter to skip.
